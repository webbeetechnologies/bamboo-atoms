# useComponentTheme Hook

We can get the theme styles of a default component using `useComponentTheme` hook.
It accept an argument called `componentName` in which we can enter the name of the component that we want to get the styles of.

Under the hood, `useComponentTheme` leverages the default `extractStyles` function. And if the user replace it with their own implementation of `extractStyles` function and returns the theme styles for the default components accordingly.
`useComponentTheme` hook is used inside the default components to fetch and apply the individual theme styles and apply them. That is why we can see the styles changes if we enter the theme styles for the default components in the theme provider.

### Example Usage

```tsx
import { useComponents, useComponentTheme, ProvideTheme, extendTheme } from '@webbee/bamboo-atoms';

// this will return a deepmarged theme obj
const theme = extendTheme({
    Heading: {
        width: 300,
        marginBottom: 50,

        dark: {
            color: '#fff',
        },
        light: {
            color: '#000',
        },
    },
});

const TextContainer = ({ componentName }: { componentName: string }) => {
    const { Text } = useComponents();
    const componentTheme = useComponentTheme(componentName);

    return (
        <Text>
            {componentName}Theme: {JSON.stringify(componentTheme)}
        </Text>
    );
};

export const Example = ({ componentName }: Props) => {
    return (
        <ProvideTheme theme={theme}>
            <TextContainer componentName={componentName} />
        </ProvideTheme>
    );
};
```
